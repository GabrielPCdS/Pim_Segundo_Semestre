# 🌱 BioCiclo: Gestão de Resíduos e Logística Reversa

O **BioCiclo** é uma plataforma tecnológica desenhada para transformar a gestão de resíduos sólidos em pequenos comércios e condomínios, substituindo processos informais por um ecossistema digital inteligente.

---

## 🎯 O Projeto
A solução foca em conectar geradores de resíduos a cooperativas de reciclagem, garantindo rastreabilidade, eficiência logística e métricas reais de impacto ambiental.

---

## 🛠️ Tecnologias Utilizadas

O desenvolvimento do BioCiclo foi fundamentado na integração de tecnologias robustas e metodologias modernas:

### 💻 Backend e Lógica
* **Linguagem C# (.NET):** Utilizada para a construção de um backend seguro e de alto desempenho.
* **Programação Orientada a Objetos (POO):** Aplicação rigorosa de conceitos como **Herança, Polimorfismo e Encapsulamento** para criar um código modular e fácil de manter.

### 🗄️ Persistência de Dados
* **Microsoft SQL Server:** SGBD relacional utilizado para garantir a integridade dos dados e a segurança das informações de usuários e registros de coleta.
* **Modelagem de Dados:** Estrutura normalizada para suportar consultas complexas e relatórios de impacto ambiental.

### ⚙️ Engenharia e Processos
* **Metodologias Ágeis (Scrum/Kanban):** Gestão do ciclo de vida do software com foco em entregas contínuas e melhoria constante.
* **Modelagem UML:** Uso de diagramas de classe e de caso de uso para o planeamento arquitetural da solução.

### 🔧 Ferramentas de Desenvolvimento
* **Visual Studio:** IDE principal para codificação e depuração.
* **Git & GitHub:** Ferramentas essenciais para o controlo de versão e colaboração técnica.

---

## 🚀 Problema e Solução

### O Problema
A informalidade na comunicação (WhatsApp/Telefone) entre geradores e cooperativas causa perda de dados, rotas de coleta ineficientes e falta de visibilidade sobre o volume reciclado.

### A Solução
Uma plataforma centralizada que oferece agendamento inteligente, dashboards de monitoramento e otimização logística baseada em dados reais.
